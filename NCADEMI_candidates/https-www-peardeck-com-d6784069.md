# Pear Deck
**Vendor:** Pear Deck Learning
**Description:** Pear Deck is an interactive technology tool that transforms presentation slides into opportunities for formative assessment and active learning. It functions as an add-on to platforms like Google Slides, enabling teachers to create engaging lessons with various question types, provide real-time feedback, and differentiate instruction.
**Product Website:** https://www.peardeck.com/

### AI Generated Insights
Description: Pear Deck demonstrates a commitment to accessibility, largely through its integration with Microsoft's Immersive Reader, which offers text-to-speech, translation, and adjustable text settings. The platform provides various accommodation features for assessments, including customizable text, high-contrast themes, and audio response options. While a 2023 VPAT claims conformance with WCAG 2.1 Level AA, external analysis notes partial conformance for freehand drawing and annotation tools, which could impact users of some assistive technologies. The vendor, Pear Deck Learning, emphasizes a rigorous, human-led testing approach in developing its VPATs, indicating a proactive stance on digital accessibility. Overall, the product aims to create inclusive learning environments by offering diverse supports for students.
