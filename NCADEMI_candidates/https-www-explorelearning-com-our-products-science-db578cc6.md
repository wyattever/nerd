# Science4Us
**Vendor:** ExploreLearning
**Description:** Science4Us is a research-based STEM program designed for K-2 students, offering interactive online activities, games, songs, virtual notebooks, and hands-on experiments to introduce foundational science topics and skills. It aligns with the 5E instructional model and aims to support diverse learning styles.

### AI Generated Insights
Science4Us prioritizes accessibility for pre- to emerging readers through its design, which heavily incorporates images alongside read-aloud text. The platform features alt text for images, offers text-to-speech capabilities via the student dashboard, and avoids reliance on color for navigation. It employs a simple, linear navigation structure and provides audio controls for both lessons and games. While keyboard input is supported in some games, certain interactive elements still require mouse input. The product ensures high color contrast ratios and makes printable text available for various content types, though some videos include embedded captions, and animated activities may occasionally lack written instructions.
