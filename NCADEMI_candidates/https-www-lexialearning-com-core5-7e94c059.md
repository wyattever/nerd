# Lexia® Core5® Reading
**Vendor:** Lexia
**Description:** Lexia® Core5® Reading is a technology-based, personalized reading program designed for students in PreK through 5th grade. It provides explicit and systematic instruction in six areas of reading, leveraging an adaptive blended learning model to deliver differentiated literacy instruction and accelerate skill development for all students.
