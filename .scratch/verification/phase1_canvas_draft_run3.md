# Canvas
**Vendor:** Instructure
**Description:** Canvas is a widely-used learning management system (LMS) designed to facilitate accessible and inclusive online learning experiences for students and educators. It provides tools and features to support diverse learning needs and is regularly updated to meet accessibility standards.
