Canvas is designed as an accessible learning management system, adhering to standards such as WCAG 2.1 AA, WCAG 2.2 AA, Section 508, and EN 301 549. It provides various built-in features and tools, including an Accessibility Checker, to help educators create and deliver accessible course content. The platform supports assistive technologies and screen readers, with resources available for best practices in accessible course design. Independent verification confirms its conformance with WCAG 2.2 Level A and AA, and official documentation, including VPATs, is available.
